Detailed Description

An immutable class containing a card's static data.

Any data that does not change during the course of the game (such as base military/political skill or fate cost) should go here. This class saves some memory by avoiding having to store bits of constant data with each card in the game. Each instance of a particular Card in the game can simply reference a CardData object for it's constant data.

The builder class should make it easier to construct a CardData object but you can use the CardGenerator class to generate the CardData object from a card id.
